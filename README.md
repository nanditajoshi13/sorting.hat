# Sorting Hat
A terminal-only C++ game that simulates the magical _Harry Potter Sorting Hat_!  
Answer **5 quiz questions**, and at the end, the Sorting Hat will assign you to one of the four Hogwarts house:
**Gryffindor, Hufflepuff, Ravenclaw, or Slytherin.** 

## Overwiew
This project is a text-based interactive interactive game inspired by the Sorting Hat ceremony from _Harry Potter_.  
Players respond to a series of multiple-choice questions, and based on their answers, 
the program calculates which house best fits their personality.

## Feature
- Immersive quiz experience with 5 questions.
- House assignment at the end (Gryffindor, Hufflepuff, Ravenclaw, Slytherin).
- Terimal-only gameplay (no GUI, pure C++ console fun).
- Replay option to try different answers and see new outcomes.

## Getting Started
### Prerequisites
- A C++ compilor (e.g. g++ etc.)
- Terminal/command prompt access.
### Compilation
g++ sorting_hat.cpp -o sorting_hat
### Run the Game
./sorting_hat

## Gameplay
1. Launch the program in your terminal.
2. Answer 5 multiple-choice questions honestly (or mischieveously!).
3. The Sorting Hat will tally your responses.
4. Discover your Hogwarts house!
